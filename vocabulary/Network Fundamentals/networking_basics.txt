1→ARPANET|The first documented network, funded by the US Defence Department in the late 1960s; precursor to the modern Internet.|networking history
2→WWW|World Wide Web; invented by Tim Berners-Lee in 1989, forming the basis of the Internet as we know it today.|networking history
3→Private Network|A small, self-contained network (e.g. a home or office LAN) not directly exposed to the public Internet.|network type
4→Public Network|The Internet; the network that connects private networks together.|network type
5→IP Address|A numerical label assigned to each device on a network, used to identify and locate it; divided into four octets.|addressing

7→IPv4|The original Internet Protocol addressing scheme; supports approximately 4.3 billion addresses.|addressing
8→IPv6|The newer IP addressing scheme supporting up to 2^128 addresses (340 trillion+); more efficient than IPv4.|addressing
9→MAC Address|Media Access Control address; a unique 12-character hexadecimal identifier assigned to a network interface at the factory.|addressing

11→MAC Spoofing|The act of faking a MAC address to impersonate another device, potentially bypassing MAC-based firewall rules.|network attack
12→ICMP|Internet Control Message Protocol; used by tools like ping to test the performance and reliability of a connection.|protocol
13→Ping|A networking tool that uses ICMP packets to test whether a connection to a device or website exists and is reliable.|networking tool
14→Octet|One of four 8-bit segments of an IPv4 address; values range from 0 to 255.|addressing
15→Firewall|A hardware or software security tool that filters network traffic by blocking unauthorized incoming and outgoing connections.|security tool
16→ISP|Internet Service Provider; the company that provides a device's public IP address and its connection to the Internet.|networking concept
17→Public IP Address|The IP address assigned by an ISP that identifies a device on the Internet; shared across all devices on the same network.|addressing
18→Private IP Address|The IP address used to identify a device within a private local network; not directly visible to the Internet.|addressing
